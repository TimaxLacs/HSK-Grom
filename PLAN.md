# План разработки сайта ЧСК "Гром"

## 1. Общая концепция и Дизайн
- **Стиль:** Строгий, милитари-эстетика, но чистая и современная.
- **Цветовая гамма:**
  - **Фон:** Однотонный темный (Dark Grey / Graphite / Deep Olive). Не фото-фон.
  - **Текст:** Белый / Светло-серый.
  - **Акцент:** Оранжевый/Желтый или фирменный цвет шеврона.
- **Типографика:**
  - Заголовки: Брутальный шрифт (Stencil / Military style).
  - Основной текст: Читабельный sans-serif (Inter / Roboto).

## 2. Структура проекта (Pages)

### Глобальные элементы (на всех страницах)
- **Header (Шапка):**
  - Логотип (Шеврон).
  - Навигационное меню: Главная, О нас, Команда, Галерея, Рекрутинг, Устав, Методичка.
  - Мобильное меню (бургер).
- **Footer (Подвал):**
  - Контакты.
  - Ссылки на соцсети.
  - **QR-код** с контактами.
  - Копирайт.

---

### Страницы

#### 1. Главная (Home)
- **Hero-секция (Верх):**
  - **Видео-фон** (зацикленное, без звука, с затемнением).
  - Заголовок: Название команды.
  - Кнопка **"Вступить"**: Открывает модальное окно (Popup).
- **Секция "О нас":**
  - Краткое текстовое описание команды.
- **Модальное окно "Вступить":**
  - Появляется поверх сайта.
  - Содержит контакты 3-х человек (Командир, Замком, Рекрутер).
  - Фото/Аватар + Имя/Позывной + Ссылка на ЛС/Телефон.

#### 2. Команда (Roster)
- **Верх:** Широкое атмосферное фото команды.
- **Структура (Иерархия):**
  - **Уровень 1:** Командир (1 карточка по центру).
  - **Уровень 2:** Замком (1-2 карточки).
  - **Уровень 3:** Старшины.
  - **Уровень 4 (Специалисты):** Столбцы по ролям (Пулеметчики, Снайперы, Медики, Саперы и т.д.).
  - **Уровень 5:** Штурмовики (Основной состав).
- **Карточка бойца:** Фото, Позывной, Звание/Роль.

#### 3. Рекрутинг (Recruiting)
- **Верх:** Широкое фото команды.
- **Контент:**
  - Основные требования (список).
  - Дополнительная информация (текстовые блоки).
  - Кнопка **"Вступить"** (вызывает то же модальное окно с контактами).

#### 4. Устав (Charter)
- **Верх:** Широкое фото (опционально) или заголовок.
- **Контент:**
  - Список правил, оформленный в виде **Аккордеона** (Раскрывающиеся списки).
  - Разделы: Общие положения, Обязанности, Запреты, Техника безопасности.

#### 5. Методичка (Training Manual)
- **Верх:** Широкое фото или заголовок.
- **Контент:**
  - Объемные блоки-карточки (Grid layout).
  - Разделы: Снаряжение (фото + описание), Тактика, Радиосвязь.
  - Кнопка/Блок: **"Скачать методичку в PDF"**.

#### 6. Галерея (Gallery)
- **Контент:** Сетка фотографий и видео.
- Возможность просмотра в полном размере (Lightbox).

---

## 3. Технический стек
- **Framework:** React + Vite.
- **Styling:** Tailwind CSS (для быстрой верстки и адаптивности).
- **Animation:** Framer Motion (для плавных появлений и модальных окон).
- **Routing:** React Router DOM.
- **Icons:** Lucide React / Heroicons.

## 4. Этапы разработки
1. **Инициализация:** Создание проекта, настройка Tailwind.
2. **Базовый Layout:** Реализация Header и Footer.
3. **Главная страница:** Видео-фон, модальное окно контактов.
4. **Страница Команды:** Верстка иерархической сетки.
5. **Страницы Устав и Методичка:** Верстка аккордеонов и карточек.
6. **Рекрутинг и Галерея:** Заполнение контентом.
7. **Адаптация:** Проверка на мобильных устройствах.
